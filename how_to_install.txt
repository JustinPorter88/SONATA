
1. Make an exact copy of the basic conda environment with the following command:
	$ conda create --clone base --name sonata 	

2. Switch to the new environement with:
	$ source activate sonata

3. Install the pythonocc-package. Use the version 17.3 because it will avoid problems with dependecies of other packages such as matplotlib/freetype and pyqt 
	$ conda conda install -c conda-forge -c dlr-sc -c pythonocc -c oce pythonocc-core==0.17.3 

4. conda install -c conda-forge pint intervaltree
5. pip install openmdao triangle shapely==1.6.4 

6. Make a small change to one file of pythoncc: src/Display/SimpleGui.py _ https://github.com/tpaviot/pythonocc-core/commit/d90e0c3fb17a0dbe3beb8ad6cd46d9516032f290
